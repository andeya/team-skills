# 风险与验证计划

> specAgent 产出 | 2026-06-23

## 风险识别

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| Mermaid 语法错误 | 图不渲染 | 低 | 使用 Mermaid Live Editor 验证 |
| 图过于复杂难读 | 用户看不懂 | 中 | 控制节点数 ≤ 25，使用子图分组 |

## 验证计划

| 验证项 | 验证命令 | 预期结果 |
|--------|----------|----------|
| Mermaid 语法 | 在线验证或本地渲染 | 无语法错误 |
| Skill 覆盖 | 对照 README Skill 清单 | 全部 12 个 Skill 在图中出现 |
| 关系正确 | 对照各 SKILL.md 的「集成关系」 | 上下游依赖正确 |

## 停下来问人的条件

1. Mermaid 图超过 30 个节点 → 停下来考虑拆分
2. 用户对图的方向/布局有意见 → 调整
