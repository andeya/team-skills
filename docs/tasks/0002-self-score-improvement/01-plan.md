# 任务规划：Team Skills 自评分改进

> specAgent 产出 | 2026-06-23

## 一、目标

### 成功标准

1. Team Skills 项目在 team-score 评分中达到 100/100 满分
2. 所有 7 项硬门槛全部通过
3. 5 个维度全部满分（25+25+27+13+10）

### 非目标

- 不修改任何 SKILL.md 的功能逻辑
- 不新增或删除 Skill
- 不改变项目架构

## 二、阶段拆分

| 阶段 | 内容 | 预期产出 | 验证方式 |
|------|------|----------|----------|
| P1: 评分 | 运行 team-score 收集证据并评分 | 评分报告 | 5 个子 Agent 扫描结果 |
| P1: 修复 | 修复所有非满分项 | 修复后的文件 | CI 模拟检查 |
| P1: 验证 | 再次评分确认改进效果 | 第二轮评分报告 | 章节完整性检查 |
| P1: 再修复 | 修复剩余问题 | 修复后的文件 | 文件检查 |
| P1: 终验 | 最终评分确认满分 | 第三轮评分报告 | 全部维度满分 |

## 三、预算

- 总预算：5 轮对话（评分→修复→评分→修复→终验）
- 每轮预算：1 次完整扫描或 1 组修复操作
