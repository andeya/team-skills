# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 0.2.x | :white_check_mark: 支持 |
| 0.1.x | :x: 不支持 |

## 报告漏洞

Team Skills 是一个提示词和规则的框架——它不直接执行任意代码或处理敏感数据。但如果你发现安全问题：

1. **不要**公开提交 Issue
2. 直接联系维护者或使用 GitHub 的私有漏洞报告功能
3. 你应在 48 小时内收到回复
4. 如果问题确认，将在 7 天内发布修复

## 关注范围

安全问题包括：

- SKILL.md 文件中的恶意提示注入
- 不安全的命令执行建议
- 模板或示例中敏感信息的暴露
- AI Agent 工作流中的社会工程风险

## 不关注范围

- 通用 AI 安全问题（对齐、偏见等）
- Claude Code、Cursor 或其他工具的漏洞
- 第三方依赖的问题
